## CSS

- static：配置方法を指定せず、場所も指定しません。(初期値)
- relative：相対位置から配置を指定します。
- absolute：絶対位置から配置を指定します。
  - relative,absolute は top,right,bottom で場所を変更
- fixed：絶対位置から配置を指定し、画面をスクロールしても固定
